# [체험단] 문토 체험단(Trial) 기능 추가

| 항목 | 내용 |
| --- | --- |
| 배경 | 첫 모임을 망설이는 유저를 위해 부담 없이 낮은 가격에 참여할 수 있는 "체험단" 기능을 출시합니다.<br><br>호스트가 초대권을 구매하여 체험단을 모집하고, 유저는 할인된 가격으로 참여 후 리뷰를 작성하면 참가비를 환급받는 구조입니다. |
| 현재 문제점 | 신규 유저의 첫 모임 참여 허들이 높아 이탈률이 높음.<br><br>호스트 입장에서는 신규 멤버 확보와 리뷰 축적이 어려움.<br><br>기존에는 얼리버드(퍼센트 할인)만 있어 적극적인 신규 유저 유치 수단이 부족함. |
| 개선 방향 | 체험단 기능 신규 개발 — 목록 노출, 상세 페이지 분기, 체험단 전용 결제/환급, 호스트 초대권 시스템, 리뷰 기반 환급, 별도 정산 로직.<br><br>기존 소셜링 로직(멤버, 결제, 환불, 정산, 알고리즘)에 영향 없이 체험단을 별도 레이어로 추가하는 방향.<br><br>소셜링 상세 페이지는 웹뷰 전환도 검토 중. |
| 고려요소 | 기존 멤버/결제/정산 로직 불간섭 원칙 (체험단 멤버 별도 테이블, 정산 별도 스케줄러).<br><br>체험단 정산은 리뷰 기한(7일) 이후 실행하여 환급 확정 후 정산.<br><br>승인제는 소셜링/체험단 각각 독립 운영.<br><br>소셜링 상세 페이지 기술 부채(Flutter 19단 중첩, God Object 뷰모델)로 인해 웹뷰 전환 검토 필요. |
| 기대효과 | 신규 유저 첫 모임 참여율 증가.<br><br>호스트 리뷰 축적 가속화.<br><br>플랫폼 초대권 판매 수익. 체험단 → 정규 멤버 전환율 확보. |
