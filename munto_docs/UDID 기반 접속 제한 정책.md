## 1. 적용 대상

이 정책은 문토 플랫폼 앱을 이용하는 전체 사용자에게 적용됩니다.

- 카운트 기준: 최근 365일 이내 로그인 이력이 있는 계정/기기만 포함, 365일 이상 이력 없는 계정/기기는 자동 제외

## 2. 정책 내용

### 2-1. 용어 정의

| 용어 | 정의 |
| --- | --- |
| UDID | 기기 고유 식별자. iOS는 IDFV, Android는 ANDROID_ID |
| 기기 공유 그룹 | 동일 기기에서 3개 이상의 계정이 로그인한 그룹. 정상적인 상황을 넘어서는 다계정 운영 정황 |
| 브릿지 계정 | 2개 이상의 기기 공유 그룹에 속한 계정. 서로 다른 그룹을 연결하는 다리 역할 |

---

### 2-2. 단일 기기 다중 계정 접속 제한 (1기기 당 최대 2계정)

**목적:** 하나의 기기에서 여러 계정을 운영하는 다중 계정 어뷰징 차단

**기준:** 1기기 최대 2계정 허용, 3번째 계정 접속 시도부터 차단

- 근거: UDID 브로커 탐지 분석 보고서 — 3개 이상 계정 공유 기기를 비정상으로 정의

**처리 방식**

- 동일 UDID의 365일 내 활성 계정이 2개인 상태에서 신규 계정 로그인/가입 시도 시 즉시 차단
- 앱 내 안내 문구 노출: "해당 기기에서는 추가 계정 접속이 불가합니다"
- 차단 이력 DB 기록 (UDID, 시도 일시)

---

### 2-3. 단일 계정 당 다중 기기 접속 제한 (1계정 당 최대 2기기)

**목적:** 하나의 계정이 다수의 기기에서 접속해 브로커 그룹 간 연결자(브릿지) 역할을 하는 행위 차단

**기준:** 1계정 최대 2기기 허용, 3번째 기기 접속 시도부터 차단

- 근거: UserDeviceLog 디바이스 사용 통계 (2026-02-25 ~ 2026-03-18, 87,006명) — 2개 이하 기기 사용자 99.2%, 3개 이상 사용자 0.8%

| 기기 수 | 유저 수 | 비율 |
| --- | --- | --- |
| 1개 | 82,766명 | 95.1% |
| 2개 | 3,575명 | 4.1% |
| 3개 이상 | 665명 | 0.8% |

**처리 방식**

- 동일 계정의 365일 내 활성 기기가 2개인 상태에서 3번째 기기 로그인 시도 시 즉시 차단
- 앱 내 안내 문구 노출: "로그인 가능한 기기 수를 초과하였습니다"
- 차단 이력 DB 기록 (account_id, UDID, 시도 일시)

### 2-4. 기존 위반 계정 소급 적용

**1순위 — 불법 채팅방 연관 기기 공유 그룹 (35개 기기, 122명)**

브로커가 운영한 불법 채팅방 소셜링 목록 34개와 탐지 데이터를 대조한 결과, 해당 소셜링에 신청한 602개 디바이스 중 35개가 기기 공유 그룹에 해당하며, 이 35개 기기에서 활동한 유저 122명.

대상 파일: `illegal_chat_high_risk_devices_20260213_122643.csv`

**2순위 — 기기 공유 그룹 전체 계정 (390명, 1순위 포함)**

전체 UDID 로그 5.92억 행 전수 스캔으로 발견된 기기 공유 그룹(686개) 관련 유저 전체입니다. 소셜링 신청 활동 여부와 무관하게 동일 기기에서 3개 이상 계정 로그인 이력이 확인된 계정으로, 수립된 정책 기준을 위반한 대상입니다. 소셜링 신청 외에도 DM 등 외부 채널을 통한 어뷰징 가능성을 배제할 수 없음.

## 3. 예외 및 유의사항

**CS 부하 분산**

- 1순위(122명) → 2순위(390명) 순서로 순차 발송해 CS 문의 집중 방지

**소셜링 폐강 리스크**

- 차단된 계정이 신청한 소셜링의 최소 인원 미달로 폐강 발생 가능
- 영구 정지 전 기결제 건 파악 시 해당 소셜링 호스트에게 사전 안내 필요

**호스트 계정 포함 가능성**

- 소급 적용 전 390명 중 호스트 계정 여부 사전 확인 후 별도 처리

**이의 제기**

- CS 채널 이의 제기 접수 시 운영팀 수동 차단 해제 가능, 해제 시 사유 및 처리자 기록 필수
- 기기 교체, 중고 구매 등 정상적인 케이스는 365일 카운트 기준 적용으로 자동 제외
