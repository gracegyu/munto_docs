## **2. 통계적 기준**

### **2.1 신뢰구간 이론**

좋아요 비율은 이항분포를 따르며, 95% 신뢰구간 폭은 다음 공식으로 계산됩니다:

```
신뢰구간 폭 = 2 × 1.96 × √(p(1-p)/n)
```

여기서:

- p = 예상 좋아요 비율 (최대 분산인 0.5 사용)
- n = 평가 횟수

## **4. 문제점 분석**

### **4.1 남성 유저의 심각한 평가 부족**

- **92.5%의 남성**이 10회 이하 평가
- 최고 평가 횟수가 28회로, ±15% 기준(43회)에도 미달
- 통계적으로 **어떤 남성 유저도 신뢰할 수 있는 점수를 가지지 못함**

### **4.2 성별 간 극심한 불균형**

- 여성은 남성보다 평균 **7.2배** 많은 평가를 받음
- 이는 남성 유저가 여성 프로필을 더 많이 평가하기 때문
- 동일한 점수 기준 적용 시 성별 간 점수 신뢰도가 크게 다름

### **4.3 현재 점수 시스템의 한계**

## **5. 결론**

현재 시스템에서 산출되는 매력도 점수는 **통계적 신뢰성이 부족**합니다.

특히 남성 유저의 경우, 충분한 평가 데이터가 축적되기 전까지 점수 기반 매칭 알고리즘의 효과가 제한적입니다.

**즉각 조치 필요 사항**:

1. 점수 전환 기준을 현실에 맞게 하향 조정 (43회 → 10~15회)
2. 성별별 차등 기준 적용 검토
3. 평가 적은 유저 우선 노출 정책 도입

**중장기 과제**:

1. 점수 신뢰도 지표 도입
2. 베이지안 추정 방식 적용 (사전 확률 + 관측 데이터)
3. 다차원 매력도 점수 시스템 설계
