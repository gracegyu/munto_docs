## **2. 버킷 시스템 (연령/지역)**

**역할 정의**

버킷은 **노출 경계가 아니라 점수 계산 시 비교 기준(reference group)**으로 사용합니다.

- **후보 생성**: 유저가 설정한 나이/거리/필터를 우선 적용
- **버킷**: 노출을 막는 기준이 아니라, **동일 연령대 내 상대적 위치**를 계산하는 용도로만 사용
- 이 구조로 유저 필터와 충돌하지 않으면서 점수 정규화 목적을 달성할 수 있습니다.

---

**2.1 버킷 분류 기준**

| 버킷 유형 | 분류 | 설명 |
| --- | --- | --- |
| 연령대 | 20~26세, 27~33세, 34~40세, 41세+ | 7년 단위로 구분 |
| 지역 | 수도권, 비수도권 | 수도권: 서울, 경기, 인천 |

---

**2.2 버킷 활용 방식 (Reference Group)**

버킷은 **점수 정규화용 reference group**으로만 사용합니다.

```
┌─────────────────────────────────────────────────────────────┐
│  버킷 활용 방식 (Reference Group)                            │
│                                                             │
│  • 후보 생성: 유저 필터(나이/거리/기타) 우선 적용 → 교차 버킷 노출 가능 |
│  • 점수 계산: 동일 버킷 내 percentile로 상대적 위치 산출       │
│  • 예: 25세 남성 점수 = "20~26세 남성들 중 상위 몇 %"         │
│  • 설계 의도: 연령대별 점수 스케일 차이를 보정해 공정한 랭킹   │
└─────────────────────────────────────────────────────────────┘
```

**설계 의도**: 유저 선호도(필터)는 그대로 반영하고, 점수 정규화 목적만 버킷으로 분리해 유지합니다. 특히 초기 규모에서는 버킷을 하드 경계로 쓰면 풀이 과도하게 축소될 수 있어, 평가 축적 속도가 느려질 수 있습니다.

---

### 2.3 나이 버킷 선택

- 유저 프로필이 생성되는 시점에 2.1에 정의된 버킷에 따라 프로필 상 나이를 기준으로 버킷을 할당합니다.
- 버킷은 **점수 정규화용 reference group**으로만 사용되며, 노출 풀을 제한하는 데에는 사용하지 않습니다.
- 유저는 추천/발견 탭에서 필터를 통해 **상대방 나이를 상세 구간으로 선택**할 수 있으며, 이는 버킷과 독립적으로 동작합니다.

## 3. 필터 시스템

### 3.1 소프트 필터 (시스템 기본값)

유저가 필터를 설정하지 않은 항목에 대해, 시스템이 통계적 선호도를 반영하여 노출 순위에 가중치를 부여합니다. 후보를 제외하지 않으며, 선호도가 높은 유저가 우선 노출됩니다.

**나이차 점수**

```
필터 미설정 시: 노출 순위 = 남or여 노출점수 × 나이차 점수
필터 설정 시: 노출 순위 = 남or여 노출점수 (나이차 점수 미적용)
```

| 연락처 차단 | 연락처 기반 지인 차단 |
| 탈퇴/정지 | status가 ACTIVE가 아닌 유저 |

> **비활성 기준 유연화**: 초기 서비스 단계에서는 유저 풀 확보를 위해 15일 기준 적용, 알고리즘 안정화 후 7일로 축소
> 
> 
> **중요**: 보여질 사람이 없는 경우, 노출 수를 줄이거나 미노출 처리
> 

### **6.2 여성이 보는 남성 노출 규칙**

### **6.2.1 신규(NEW) 여성이 보는 남성 (인터랙션 15회 미만)**

```
┌─────────────────────────────────────────────────────────────┐
│  신규 여성 → 기존 남성만 노출 (신규/정착중 남성 X)                    │
│                                                             │
│  • 7일 이내 활동한 남성만 (풀 부족 시 15일까지 확장)                  │
│  • 매력도 상위 0~30% 구간에서 확률 함수로 선택                       │
│  • 신규/정착중 남성은 절대 노출되지 않음                             │
└─────────────────────────────────────────────────────────────┘
```

| **여성 분류** | **노출 대상 남성** | **노출 범위** |
| --- | --- | --- |
| 신규 (인터랙션 <15) | 기존 남성만 | 상위 0~30% 구간, 확률 함수 적용 |

### **6.2.2 정착 중(SETTLING) 여성이 보는 남성 (인터랙션 15회+ AND D+7 이내)**

```
┌─────────────────────────────────────────────────────────────┐
│  정착 중 여성 → 기존 남성만 노출 (신규/정착중 남성 X)                  │
│                                                             │
│  • 7일 이내 활동한 남성만 (풀 부족 시 15일까지 확장)                  │
│  • 상위 0~30% 구간에서 시작, 자기 점수 구간까지 점진적 확대             │
│  • 평가 충족되었으므로 실제 매력도 점수 반영 시작                      │
│  • 신규/정착중 남성은 절대 노출되지 않음                             │
└─────────────────────────────────────────────────────────────┘
```

**점진적 노출 범위 확대:**

15회 평가 충족 후, D+7이 될 때까지 상위 구간에서 실제 점수 구간으로 점진적으로 이동합니다.

| **조건** | **노출 범위** | **설명** |
| --- | --- | --- |
| 15회 평가 충족 직후 | 상위 0~30% | 상위 남성 풀 유지 |
| 15회 충족 ~ D+7 도달 전 | 상위 0~30% → 자기 점수 구간 | **D+7까지 남은 기간에 걸쳐** 점진적 이동 |
| D+7 도달 | 자기 점수 ± 편차 | 실제 매력도 기반 노출 (기존 여성 전환) |

> **핵심 기준**: 이동 시작은 "15회 평가 충족" 시점이며, D+7이 될 때까지 점진적으로 이동
> 
> 
> **예시**: D+3에 15회 충족 시, D+3~D+7 (4일간) 점진 이동 / D+5에 15회 충족 시, D+5~D+7 (2일간) 점진 이동
> 
> **설계 의도**: 평가가 충족된 후에는 자기 점수에 맞는 상대를 점진적으로 보여줌 (급격한 변화 방지)
> 

| **여성 분류** | **노출 대상 남성** | **노출 범위** |
| --- | --- | --- |
| 정착 중 (인터랙션 ≥15, D+7 이내) | 기존 남성만 | 상위 0~30% → 자기 점수 구간으로 점진적 이동 |

### **6.2.3 기존(ESTABLISHED) 여성이 보는 남성 (인터랙션 15회+ AND D+7 이후)**

```
┌─────────────────────────────────────────────────────────────┐
│  기존 여성 → 기존 남성 + 신규/정착중 남성 (버킷 분리)                  │
│                                                             │
│  • 7일 이내 활동한 남성만 (풀 부족 시 15일까지 확장)                  │
│  • 매력도 기준 노출, 실제 평가 점수에 따라 범위 확대                   │
│  • 신규/정착중 남성은 버킷 비율로 노출 (아래 참조)                     │
└─────────────────────────────────────────────────────────────┘
```

### **추천/발견 각 4명 구성 (기존 여성 기준)**

| **탭** | **기존 남성** | **정착중 남성** | **신규 남성** | **비고** |
| --- | --- | --- | --- | --- |
| 추천 (4명) | 3명 | 0~1명 (5~14회) | 0~1명 (0~4회) | SETTLING 또는 NEW 중 택1 |
| 발견 (4명) | 3명 | 0~1명 (5~14회) | 0~1명 (0~4회) | SETTLING 또는 NEW 중 택1 |

### **정착중/신규 남성 노출 비율 규칙**

| **기준** | **규칙** | **설명** |
| --- | --- | --- |
| 1회 사이클 기준 | 최대 1명 (25%) | 4명 중 신규/정착중 남성은 최대 1명 |
| 전체 풀 분배 기준 | 평균 25% 목표 | 신규/정착중 남성 풀을 여성들에게 고르게 분배 |

> **개인별 vs 전체 풀 관점**:
> 
> - **개인별**: 한 여성이 하루 3사이클 모두에서 신규/정착중 남성 1명씩 볼 수도 있음 (최대 3명/12명)
> - **전체 풀**: 신규/정착중 남성 200명을 여성 100명에게 분배 시, 일부 여성은 매 사이클 노출, 일부는 미노출
> - **목표**: 전체적으로 신규/정착중 남성의 평가 기회를 고르게 분산
> 
> 발견 탭 노출 로직은 추천 탭과 동일한 로직으로 적용
> 

### **6.3 남성이 보는 여성 노출 규칙**

### **6.3.1 신규/정착중 남성이 보는 여성 (인터랙션 15회 미만)**

```
┌─────────────────────────────────────────────────────────────┐
│  신규/정착중 남성 → 기존 여성만 노출 (신규/정착중 여성 X)               │
│                                                             │
│  • 7일 이내 활동한 여성만 (풀 부족 시 15일까지 확장)                  │
│  • 여성 랭킹 기준: 매력도, 접속빈도, 좋아요보낸비율                    │
│  • 신규/정착중 여성은 절대 노출되지 않음                             │
└─────────────────────────────────────────────────────────────┘
```

| **남성 분류** | **노출 대상 여성** | **여성 랭킹 기준** |
| --- | --- | --- |
| 신규 (인터랙션 0~4회) | 기존 여성만 | 여성 점수 기반 |
| 정착 중 (인터랙션 5~14회) | 기존 여성만 | 여성 점수 기반 |

### **신규 남성 티어 시스템**

신규 남성에게 노출할 여성 구간을 4단계 티어 + 노출 제외로 구분하고, 받은 시그널에 따라 티어를 조정합니다.

### **용어 정의**

| **용어** | **정의** |
| --- | --- |
| **첫 노출** | 신규 남성이 5**명의 여성에게 최초로 노출된 시점** (배치 X, 단일 노출 기준) |
| **순점수** | 긍정 인터랙션 가중치 합계 - 부정 인터랙션 가중치 합계 (신규 남성 전용) |
| **평균 점수** | 기존 남성(15회+)에게 적용되는 방식, 부정은 0에 가깝게 처리 후 평균 계산 |

### **첫 노출 관문**

```
┌─────────────────────────────────────────────────────────────┐
│  첫 노출 결과에 따른 분기                                         │
│                                                             │
│  첫 노출 → 부정 시그널 (3인 모두 프로필 미조회, 패스)                  │
│         → 즉시 노출 제외 (하위 15% 격리)                         │
│                                                             │
│  첫 노출 → 긍정 시그널 (프로필 조회 이상)                          │
│         → 평가 계속, 점수 누적으로 티어 결정                       │
└─────────────────────────────────────────────────────────────┘
```

> **설계 의도**: 첫 노출에서 프로필조차 열어보지 않으면 매력이 없는 것으로 판단, 즉시 노출 제외
> 
> 
> **순점수 vs 평균 점수 차이**:
> 
> - **신규 (순점수)**: 부정 인터랙션이 마이너스로 계산되어 점수가 하락 → 빠른 티어 조정
> - **기존 (평균)**: 부정 인터랙션이 0에 가깝게 처리되어 평균에 큰 영향 없음 → 안정적 점수 유지

### **티어 구성 (4단계 + 노출 제외)**

| **티어** | **노출 여성 구간** | **예상 분포** | **설명** |
| --- | --- | --- | --- |
| 1 티어 | 상위 0~20% 여성 | **10%** | 최상위 여성 노출 |
| 2 티어 | 상위 20~40% 여성 | **20%** | 상위 여성 노출 |
| 3 티어 | 상위 40~60% 여성 | **30%** | 중위 여성 노출 |
| 4 티어 (시작) | 상위 60~85% 여성 | **25%** | 중하위 여성 노출 **(기본 시작 티어)** |
| **노출 제외** | 노출 안 함 | **15%** | **하위 15% - 7번 매력도 구간 제한 적용** |

> **시작 티어**: 신규 남성은 첫 노출 통과 후 4 티어에서 시작
> 
> 
> **티어↔여성 점수 매핑**: 동일 백분위 매핑 적용 (예: 1티어 남성 = 상위 0~20% 여성)
> 

### **티어 점수 계산 (순점수 방식)**

신규 남성(15회 미만)은 인터랙션 가중치를 **순점수**로 계산하여 티어를 결정합니다.

```
티어 점수 = Σ(긍정 인터랙션 가중치) - Σ(부정 인터랙션 가중치)
```

### **순점수 vs 비율(평균) 방식 차이**

| **구분** | **순점수 (신규 남성)** | **비율/평균 (여성, 기존 남성)** |
| --- | --- | --- |
| 계산 방식 | 긍정 합계 - 부정 합계 | 전체 인터랙션 대비 긍정 비율 |
| 부정의 영향 | **마이너스로 직접 감점** | 0에 가깝게 처리, 평균에 큰 영향 없음 |
| 초기 이벤트 영향력 | **매우 큼** (1~2개로 티어 결정) | 작음 (누적될수록 희석) |
| 적용 목적 | 빠른 필터링 (매력 없는 유저 조기 격리) | 안정적인 점수 유지 |

> **설계 의도**: 신규 남성은 첫 몇 번의 인터랙션으로 빠르게 평가하여, 매력 없는 유저를 조기에 격리
> 
> 
> **예시**:
> 
> - 순점수: 첫 노출에서 "프로필 미조회"(-1.0) 받으면 즉시 -1.0 → 노출 제외 가능
> - 비율: 첫 노출에서 부정 받아도 전체 평균에서 희석되어 큰 변화 없음

> **인터랙션별 가중치**: 1.1 "인터랙션 종류 및 가중치" 참조
> 

### **점수 구간별 티어 매핑**

상단 티어 구성 (4단계 + 노출 제외) 섹션 참조

### **추천/발견 각 4명 구성 (기존 남성 기준)**

| 전체 풀 분배 기준 | 평균 75% 목표 | 신규/정착중 여성 풀을 남성들에게 고르게 분배 |

> **참고**: 개인별 vs 전체 풀 관점의 상세 설명은 5.2.3 "정착중/신규 남성 노출 비율 규칙" 참조

### **신규/정착중 여성 우선 노출 설계 의도**

| **이점** | **설명** |
| --- | --- |
| 신규 여성 정착 | 기존 남성에게 노출되어 좋아요를 받으면 빠르게 정착 |
| 기존 남성 경험 | 새로운 프로필을 볼 기회 제공 |
| 평가 분산 | 신규 여성 평가를 기존 남성이 담당 |

> **주의**: 신규/정착중 여성 풀이 부족할 경우 기존 여성으로 대체

> **중요**: 신규/정착중 남성 풀이 부족할 경우, 해당 슬롯은 기존 남성으로 대체 **노출 비율**: 5.2.3의 정착중/신규 남성 노출 비율 규칙 참조
> 

### **7.4 발견 큐 변경**

## **8. 매력도 구간 제한 (하위 15% 남성 격리 정책)**

하위 15% 남성은 **양방향으로 격리**됩니다. 여성에게 노출되지 않으며, 이들에게도 상위 85% 여성은 보여주지 않습니다.

```
┌─────────────────────────────────────────────────────────────┐
│  하위 15% 남성 격리 정책 (양방향)                              │
│                                                             │
│  1. 여성이 보는 남성 카드에 하위 15% 남성 미포함                 │
│  2. 하위 15% 남성에게는 하위 15% 여성만 노출                    │
│  3. 풀 부족 시 카드 수 축소 (범위 확대 금지)                    │
└─────────────────────────────────────────────────────────────┘
```

### **노출 제외 조건**

| **조건** | **처리** | **설명** |
| --- | --- | --- |
| 첫 노출 부정 시그널 | 즉시 노출 제외 | 프로필 미조회, 패스 시 바로 하위 15%로 격리 |
| 티어 점수 -3.0 미만 | 노출 제외 | 첫 노출 통과 후에도 부정 누적 시 제외 |

### **양방향 노출 규칙**

### **여성 → 남성 (여성이 보는 남성 카드)**

| **규칙** | **설명** |
| --- | --- |
| 하위 15% 남성 미포함 | 여성에게 노출되는 남성 카드에는 **하위 15% 남성이 포함되지 않음** |
| 적용 대상 | 모든 여성 (신규/정착중/기존 무관) |
| 적용 탭 | 추천 탭, 발견 탭 모두 |

> **의도**: 여성 경험 보호를 위해 매력도 낮은 남성을 아예 노출하지 않음
> 

### **남성 → 여성 (하위 15% 남성이 보는 여성 카드)**

| **규칙** | **설명** |
| --- | --- |
| 하위 15% 여성만 노출 | 노출 제외된 남성에게는 **하위 15% 여성만** 보여줌 |
| 상위 85% 여성 미노출 | 하위 15%보다 나은 여성은 **절대 노출하지 않음** |

> **의도**: 하위 15% 남성이 상위 여성에게 접근하는 것을 차단하여 전체 서비스 품질 유지
> 

### **풀 부족 시 대응**

| **상황** | **대응** | **비고** |
| --- | --- | --- |
| 하위 15% 여성 풀 부족 | 카드 수 축소 (4명→2명→0명) | 모자라면 카드 수를 줄임 |
| 카드 수 축소로도 부족 | 미노출 처리 | 해당 사이클 카드 0장 |
| **금지 사항** | 범위 확대 금지 | 상위 여성을 보여주지 않음 |

> **설계 의도**: 양이 부족하더라도 **질을 유지**. 하위 15% 남성에게 상위 여성을 노출하면 상위 여성의 경험이 저하되므로, 카드 수를 줄이더라도 범위는 유지
> 

### **프로필 변경 시 처리**

> 프로필 변경 시 처리 규칙은 **10번 "유저 상태 변경 시 처리 규칙"** 참조
> 
> - 최근 10회 평가만 유지, 정착중(SETTLING) 상태로 전환
> - 노출 제외 남성도 프로필 변경 시 정착중으로 복귀 가능

## **10. 활동 필터 세부 규칙**

### **10.1 미접속 유저 제외**

| **미접속 기간** | **처리** |
| --- | --- |
| 7일 이상 | **노출에서 제외** |
| 7일 미만 | 정상 노출 |

### **10.2 풀 부족 시 대응**

풀이 부족할 경우, 자동 확장 없이 알림 및 카드 축소로 대응합니다.

```
IF 노출 대상 풀 < 필요 노출 수 THEN
  1. 운영팀에 알림 발송 (버킷별 풀 부족 알림)
  2. 노출 카드 수 축소 (8명 → 6명 → 4명 → 2명)
  3. 최후에도 부족하면 미노출 처리
END IF
```

**절대 하지 않는 것:**

| **금지 사항** | **이유** |
| --- | --- |
| 점수 범위 확대 | 신규 유저에게 전체 유저가 랜덤 노출 → 득보다 실 |
| 자동 버킷 확장 | 매칭 품질 저하 |

**설계 목표:**

> 유저에게 "좋은" 프로필을 선별하여 보여주는 것이 목적입니다. **양이 아닌 질이 우선**되어야 합니다.

## **11. 유저 상태 변경 시 처리 규칙**

유저 상태가 변경되었을 때 매력도 점수 및 분류 처리 방식입니다.

| **상태 변경** | **처리 방식** | **설명** |
| --- | --- | --- |
| 비활성화 후 해제 | **점수 그대로 유지** | 기존 매력도 점수 및 분류 유지 |
| 탈퇴 후 재가입 | **신규로 처리** | 새로운 유저로 간주, 처음부터 평가 시작 |
| 프로필 변경 | **정착중 유저로 전환** | 최근 10회 평가만 유지, 정착중(SETTLING) 상태로 전환 |

### **프로필 변경 시 재평가 로직**

프로필을 변경하면 기존 평가를 일부 유지하면서 정착중 유저로 전환됩니다. (남녀 공통 적용)

### **처리 방식 (공통)**

| **항목** | **처리** | **설명** |
| --- | --- | --- |
| 평가 데이터 | 최근 10회만 유지 | 오래된 평가는 삭제, 최근 반응만 반영 |
| 유저 분류 | 정착중(SETTLING)으로 전환 | 5~14회 구간 유저와 동일한 취급 |

### **성별별 노출 범위 규칙**

| **성별** | **프로필 변경 후 상태** | **노출 범위** |
| --- | --- | --- |
| 남성 | 정착중(SETTLING) 전환 | 티어 기반 여성 노출 (5.3.1 규칙 적용) |
| 여성 | **D+4 유저로 간주** | 3일간 점진적 이동 (상위 0~30% → 자기 점수 구간) |

### **여성 프로필 변경 시 점진적 노출 범위 이동**

여성은 프로필 변경 후 D+4 유저로 간주되어, 3일간 점진적으로 자기 점수 구간으로 이동합니다.

| **경과 일수** | **노출 범위** | **설명** |
| --- | --- | --- |
| 변경 직후 (D+4 가정) | 상위 0~30% | 상위 남성 풀 유지 |
| 변경 후 1~2일 | 상위 0~30% → 자기 점수 구간 | 점진적으로 자기 점수 구간으로 이동 |
| 변경 후 3일 (D+7 도달) | 자기 점수 ± 편차 | 실제 매력도 기반 노출 |

> **예시**: 기존 여성(점수 상위 50%)이 프로필 변경 시
> 
> - 변경 직후: 상위 0~30% 남성 노출 (D+4 유저 취급)
> - 1~2일차: 점진적으로 상위 50% 구간으로 이동
> - 3일차: 자기 점수 기반 노출 (상위 40~60% 남성)

### **적용 대상**

| **유저 유형** | **적용 여부** | **비고** |
| --- | --- | --- |
| 신규 남성 | 적용 | 기존 평가 10회 미만이면 전부 유지 |
| 기존 남성 | 적용 | 최근 10회로 재계산 |
| 노출 제외 남성 | 적용 | 정착중으로 복귀하여 재평가 |
| 신규 여성 | 적용 | 기존 평가 10회 미만이면 전부 유지, D+4로 간주 |
| 기존 여성 | 적용 | 최근 10회로 재계산, D+4로 간주하여 3일간 점진적 이동 |

### **설계 의도**

| **이점** | **설명** |
| --- | --- |
| 과거 평가 반영 | 완전 리셋이 아닌 최근 반응 기반 재평가 |
| 빠른 적응 | 정착중 상태로 5회만 추가 평가받으면 기존 유저로 전환 |
| 남용 방지 | 기존 평가 일부 유지로 반복 리셋 남용 방지 |
| 여성 경험 유지 | 프로필 변경 후에도 상위 남성을 일정 기간 볼 수 있음 |

> **제한 사항**: 프로필 변경 후 재평가는 7일 내 1회만 가능
> 
> 
> **설계 의도**: 프로필 개선 후 다시 기회를 주되, 최근 평가를 유지하여 과도한 리셋 남용 방지
> 

### **일반 원칙**

> **설계 의도**: 정지/해제는 일시적 조치이므로 기존 평가를 유지하고, 탈퇴 후 재가입은 완전히 새로운 시작으로 처리
>

## **12. 요약: 노출 매트릭스**

### **12.1 신규/기존 교차 노출 규칙 (사분면)**

신규 유저끼리는 서로 볼 수 없고, 기존 유저를 통해서만 노출됩니다.

```
┌─────────────────────────────────────────────────────────────┐
│                        남성                                 │
│              신규/정착중          기존                       │
│         ┌──────────────┬──────────────┐                    │
│  신규/  │              │              │                    │
│  정착중 │      ✕       │      ✓       │                    │
│         │  서로 못봄    │  여성→남성    │                    │
│ 여 ─────┼──────────────┼──────────────┤                    │
│ 성      │              │              │                    │
│  기존   │      ✓       │      ✓       │                    │
│         │  남성→여성    │   서로 봄     │                    │
│         └──────────────┴──────────────┘                    │
└─────────────────────────────────────────────────────────────┘
```

| **조합** | **노출 가능 여부** | **설명** |
| --- | --- | --- |
| 신규 남성 ↔ 신규 여성 | ✕ | 서로 볼 수 없음 |
| 신규 남성 → 기존 여성 | ✓ | 신규 남성은 기존 여성만 볼 수 있음 |
| 기존 남성 → 신규 여성 | ✓ | 신규 여성은 기존 남성만 볼 수 있음 |
| 기존 남성 ↔ 기존 여성 | ✓ | 서로 볼 수 있음 |

> **설계 의도**: 신규 유저는 검증된 기존 유저에게만 노출되어 첫인상을 좋게 유지. 기존 유저는 신규 유저 평가를 담당.
> 

### **12.2 여성이 보는 남성 (상세)**

| **여성 분류** | **조건** | **볼 수 있는 남성** | **노출 범위** |
| --- | --- | --- | --- |
| 신규 | 인터랙션 <15 | 기존 남성만 | 상위 0~30% 고정, 7일 활동 |
| 정착 중 | 인터랙션 ≥15, D+7 이내 | 기존 남성만 | 상위 0~30% → 자기 점수 구간 점진 이동 |
| 기존 | 인터랙션 ≥15, D+7 이후 | 기존 75% + 신규/정착중 25% | 자기 점수 구간, 7일 활동 |

### **12.3 남성이 보는 여성 (상세)**

| **남성 분류** | **볼 수 있는 여성** | **노출 범위** | **비고** |
| --- | --- | --- | --- |
| 신규 | 기존 여성만 | 매력도+활동성+좋아요비율 | - |
| 정착 중 | 기존 여성만 | 매력도+활동성+좋아요비율 | - |
| 기존 | 기존 + 신규/정착중 여성 모두 | 매력도+활동성+좋아요비율 | **신규 여성 우선 노출** |
| 노출 제외 (하위 15%) | **하위 15% 여성만** | - | 풀 부족 시 카드 축소 |

> **신규 여성 우선 노출**: 기존 남성에게 신규/정착중 여성을 우선 배정하여 신규 여성의 정착 지원 (5.3.2 참조)
>
