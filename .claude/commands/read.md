# 📖 Notion 문서 읽기

노션 문서 링크나 ID를 통해 문서 내용을 읽고 요약해주는 명령어입니다.

**사용법**: `/read [노션 링크 또는 페이지 ID]`

**입력값**: $ARGUMENTS

## ⚠️ 중요 제약사항 ⚠️
입력값 ($ARGUMENTS)이 다음 중 하나가 아닌 경우:
- 노션 페이지 링크 (https://www.notion.so/...)
- 노션 페이지 ID (32자리 문자열)
- 빈 값 (사용법 안내)

다음을 수행해야 합니다:
1. "올바른 노션 링크나 페이지 ID를 입력해주세요. 예: https://www.notion.so/page-id 또는 32자리 페이지 ID"만 출력
2. 즉시 중단

## Step 1: 입력값 검증 및 파싱

### 1.1 입력값 형태 확인
- 빈 값인 경우: 사용법 안내 출력 후 종료
- URL 형태인 경우: 노션 링크에서 페이지 ID 추출
- 32자리 문자열인 경우: 페이지 ID로 사용

### 1.2 페이지 ID 형식 검증
- 32자리 영숫자 문자열 확인 (하이픈 포함/제외 모두 허용)
- 형식이 맞지 않는 경우 오류 메시지 출력

## Step 2: 노션 문서 읽기

### 2.1 페이지 메타데이터 조회
- `mcp_Notion_notion_retrieve_page` 도구 사용
- 페이지 제목, 생성일, 수정일 등 기본 정보 확인

### 2.2 페이지 내용 읽기
- `mcp_Notion_notion_retrieve_block_children` 도구 사용
- format을 'markdown'으로 설정하여 읽기 쉬운 형태로 조회

## Step 3: 내용 분석 및 요약

### 3.1 문서 구조 파악
- 제목, 섹션, 하위 블록 구조 분석
- 주요 내용 요소 식별 (텍스트, 리스트, 표, 이미지 등)

### 3.2 핵심 내용 추출
- 문서의 주요 목적과 내용 파악
- 중요한 정보나 액션 아이템 식별
- 데이팅 패키지 개발과 관련된 내용 강조

## Step 4: 결과 출력

### 4.1 문서 요약 형식
다음 형식으로 결과를 출력합니다:

```markdown
# 📄 노션 문서 요약

## 📋 기본 정보
- **제목**: [문서 제목]
- **페이지 ID**: [페이지 ID]
- **마지막 수정**: [수정일]

## 📝 주요 내용
[문서의 핵심 내용을 요약]

## 🎯 주요 포인트
- [중요한 포인트 1]
- [중요한 포인트 2]
- [중요한 포인트 3]

## 💡 액션 아이템 (있는 경우)
- [해야 할 일 1]
- [해야 할 일 2]

## 🔗 관련 링크나 참조
[문서 내 언급된 다른 링크나 참조사항]
```

### 4.2 오류 처리
- 페이지에 접근할 수 없는 경우: 권한 확인 안내
- 페이지가 존재하지 않는 경우: ID 확인 안내
- API 오류 발생 시: 일반적인 오류 메시지 출력

## 사용 예시

```bash
# 노션 링크로 읽기
/read https://www.notion.so/workspace/page-title-1234567890abcdef

# 페이지 ID로 읽기  
/read 1234567890abcdef1234567890abcdef

# 하이픈 포함 페이지 ID로 읽기
/read 12345678-90ab-cdef-1234-567890abcdef
```

## 주의사항

- 노션 API 토큰이 올바르게 설정되어 있어야 합니다
- 읽으려는 페이지에 Integration이 연결되어 있어야 합니다
- 개인정보가 포함된 문서의 경우 요약 시 주의해야 합니다
- 긴 문서의 경우 주요 내용만 요약하여 제공합니다
