| 구분        | 예상 비용     | 비고                                             |

| 외부 협력   | 없음          |                                                  |
| 라이선스/툴 | 약 200,000원  | Figma 유료 플랜 1개월 사용 예정                  |
| 기타        | 없음          |                                                  |
| 합계        | 20,2000,000원 |                                                  |
