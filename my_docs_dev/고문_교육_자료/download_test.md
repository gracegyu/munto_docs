# Notion 문서 업로드

이 명령어는 프로젝트의 마크다운 문서를 회사 Notion 워크스페이스에 자동으로 업로드합니다.

## 작업 흐름

1. 사용자에게 업로드할 마크다운 파일 경로를 물어봅니다
2. 해당 파일을 읽어옵니다
3. Notion에 업로드할 상위 페이지 ID를 물어봅니다 (선택사항)
4. Notion MCP를 사용하여 새 페이지를 생성합니다
5. 마크다운 내용을 Notion 블록으로 변환하여 추가합니다
6. 생성된 페이지 URL을 사용자에게 알려줍니다

## 실행 절차

### Step 1: 파일 정보 수집

사용자에게 다음을 질문:

- 어떤 마크다운 문서를 업로드하시겠습니까? (파일 경로를 입력하세요)
- Notion의 어느 페이지 하위에 생성하시겠습니까? (페이지 ID 입력, 없으면 루트에 생성)

### Step 2: 파일 읽기

`read_file` 도구를 사용하여 지정된 마크다운 파일을 읽습니다.

### Step 3: Notion 페이지 생성

Notion MCP를 사용하여:

1. 파일명을 제목으로 하는 새 페이지 생성
2. 마크다운 내용을 Notion 형식으로 변환
3. 생성된 페이지에 내용 추가

### Step 4: 결과 보고

- 성공: ✅ Notion 페이지가 생성되었습니다: [페이지 URL]
- 실패: 오류 메시지와 함께 문제 해결 방법 제안

## 주의사항

- 이미지나 첨부파일이 있는 경우 별도로 처리가 필요할 수 있음을 안내
- Notion API 제한사항 (페이지 크기, 블록 수 등)을 고려
- 마크다운 형식이 Notion에서 완벽하게 변환되지 않을 수 있음을 안내

## 예시 사용

사용자: claude upload

AI: 어떤 마크다운 문서를 업로드하시겠습니까?

사용자: my_docs_dev/개발_자료_보고서/i18n brain storming.md

AI: Notion의 어느 페이지 하위에 생성하시겠습니까? (페이지 ID 입력, 없으면 엔터)

사용자: [페이지 ID 또는 엔터]

AI: [파일 읽기 → Notion 페이지 생성 → URL 반환]

✅ Notion 페이지가 생성되었습니다: https://notion.so/munto/...
